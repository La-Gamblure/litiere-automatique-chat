## Design et fabrication

Le PETKIT Pura Max 2 adopte un design cylindrique fermé, avec une coque en ABS blanc mat qui s'intègre bien dans un intérieur moderne. L'entrée latérale est large — suffisante pour un chat de 8 kg sans qu'il se sente à l'étroit. Dimensions : 60 x 58 x 62 cm, 13,5 kg à vide.

La trappe de nettoyage se soulève facilement pour accéder au tambour. Le bac à déchets coulissant à l'arrière s'extrait d'une main. Les matériaux sont solides, les jointures propres. Le spray N50 anti-odeur est logé dans un compartiment dédié en haut de l'appareil.

Point notable : le système xSecure avec ses multiples capteurs (poids, infrarouge, anti-pincement) encercle l'intégralité de la zone de rotation. C'est le dispositif de sécurité le plus complet de notre comparatif.

## Fonctionnement au quotidien

Après le départ du chat, le tambour effectue une rotation pour tamiser la litière. Les agglomérats tombent dans le bac inférieur. Le cycle dure environ 3 minutes. Le délai de déclenchement est configurable via l'app (1, 5, 10, 15 minutes).

La capacité de 76L est généreuse. Pour un chat, le bac à déchets tient une dizaine de jours. Pour deux chats, comptez 5-6 jours. Le mode multi-chats identifie chaque animal par son poids et génère des statistiques individuelles — un vrai plus pour les foyers avec plusieurs félins.

Le spray désodorisant N50 se déclenche automatiquement après chaque cycle. L'efficacité est réelle : les odeurs sont neutralisées, pas simplement masquées. Les recharges durent environ 3 mois pour un usage normal.

## App PETKIT et connectivité

L'app PETKIT (iOS/Android) offre un contrôle complet :

- **Tableau de bord** : état en temps réel, niveau de litière et de déchets
- **Historique d'utilisation** : fréquence et durée des visites par chat
- **Mode multi-chats** : identification par le poids, statistiques individuelles
- **Programmation** : cycles planifiés, mode nuit, mode vacances
- **Alertes** : bac plein, niveau de litière bas, erreur moteur

La connexion Wi-Fi 2.4 GHz fonctionne bien. L'app est traduite en français, avec une interface claire. Seul bémol : les notifications peuvent être un peu envahissantes par défaut — pensez à les configurer. Pour un comparatif détaillé des apps, consultez notre [guide des litières connectées](/guides/litiere-automatique-connectee-app/).

## Niveau sonore

Le PETKIT Pura Max 2 fonctionne à 52 dB, soit le niveau d'une conversation normale. C'est correct sans être exceptionnel. Le bruit est régulier pendant les 3 minutes du cycle, principalement un ronronnement mécanique.

Le mode nuit réduit légèrement la vitesse de rotation et suspend les alertes sonores. Si la litière est dans une chambre, le bruit reste perceptible pendant le cycle.

## Entretien

L'entretien est bien pensé :

- **Bac à déchets** : vidange tous les 5-10 jours selon le nombre de chats
- **Tambour** : nettoyage mensuel à l'eau tiède
- **Spray N50** : recharge tous les 3 mois (~15 €)
- **Capteurs** : essuyage occasionnel si la litière y adhère

Le tambour se retire en quelques secondes pour un nettoyage complet. Les capteurs xSecure sont accessibles sans outil. Notre [guide d'entretien](/guides/litiere-automatique-entretien-nettoyage/) détaille la procédure complète. Le coût mensuel total : litière agglomérante (15-20 €) + sacs (5 €) + recharge N50 (5 €/mois amorti).

## Verdict détaillé

Le PETKIT Pura Max 2 est le couteau suisse des litières automatiques. Son système anti-odeur N50 est unique sur le marché, sa sécurité xSecure est la plus complète, et son mode multi-chats avec identification individuelle est un vrai différenciateur.

Le prix (399-599 € selon les promotions) le place en milieu de gamme. C'est nettement moins cher qu'un Litter-Robot 4 pour des fonctionnalités comparables, voire supérieures sur certains points (spray anti-odeur, sécurité).

Le principal compromis : le niveau sonore (52 dB) est correct mais pas aussi discret que les modèles premium ultra-silencieux. Et la capacité max chat de 8 kg exclut les très gros gabarits.
