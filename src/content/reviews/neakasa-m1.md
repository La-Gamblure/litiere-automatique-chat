## Design et fabrication

Le Neakasa M1 se présente comme un cylindre blanc épuré avec une ouverture frontale extra-large. C'est son point fort n°1 : l'entrée est dimensionnée pour les grands chats, y compris les Maine Coon jusqu'à 10 kg. Dimensions globales : 58 x 56 x 66 cm, 10,8 kg à vide.

La qualité de fabrication est correcte pour le segment milieu de gamme. Le plastique ABS est solide, le mécanisme de rotation est fluide. Neakasa inclut un tapis anti-projections et un lot de sacs jetables dans la boîte — un geste appréciable qui évite un achat supplémentaire immédiat.

Le design reste sobre. Pas de fioritures ni de formes fantaisistes. Le bac à déchets est accessible par l'arrière, avec un système de sac standard.

## Fonctionnement au quotidien

Le Neakasa M1 utilise un mécanisme de rotation classique : le tambour tourne après le départ du chat, tamise la litière et dépose les agglomérats dans le bac inférieur. Le cycle dure environ 2 minutes 30.

La capacité de 65L est dans la moyenne haute. Pour un chat seul, comptez une vidange tous les 7-8 jours. Pour deux chats, 4-5 jours. La garantie 2 ans pièces est un argument rassurant pour un produit de cette gamme de prix.

L'ouverture extra-large change réellement l'expérience pour les propriétaires de grands chats. Là où d'autres modèles forcent un chat de 8+ kg à se contorsionner, le Neakasa M1 offre un accès confortable.

## App et connectivité

L'app Neakasa (iOS/Android) est fonctionnelle mais basique :

- **Contrôle à distance** : déclenchement manuel, pause, remise à zéro
- **Historique** : nombre de cycles, fréquence d'utilisation
- **Notifications** : bac plein, erreur
- **Programmation** : mode nuit, délai post-utilisation

Pas de suivi santé, pas d'identification multi-chats, pas de statistiques avancées. L'app fait le minimum nécessaire, sans plus. La connexion Wi-Fi 2.4 GHz est stable.

Pour qui l'app minimaliste suffit amplement : les propriétaires d'un ou deux chats qui veulent juste savoir quand vider le bac et pouvoir déclencher un cycle à distance.

## Niveau sonore

C'est le point fort majeur du Neakasa M1 : 45 dB en fonctionnement. C'est le modèle le plus silencieux de notre comparatif, plus discret qu'un réfrigérateur.

Le bruit se résume à un léger ronronnement pendant la rotation. Aucun claquement, aucun grincement. Pour les foyers où la litière est dans un espace de vie ou une chambre, c'est un critère décisif.

Les chats les plus craintifs n'y prêtent aucune attention après un ou deux cycles d'adaptation.

## Entretien

L'entretien est standard et sans surprise :

- **Bac à déchets** : vidange hebdomadaire (1 chat) ou tous les 4-5 jours (2 chats)
- **Tambour** : nettoyage mensuel à l'eau savonneuse
- **Tapis anti-projections** : inclus, lavable en machine

Pas de consommable propriétaire. La litière agglomérante classique suffit. Le tapis inclus réduit les projections de granulés autour de la machine — un accessoire souvent vendu séparément ailleurs.

Coût mensuel : litière agglomérante (15-20 €) + sacs standard (3-5 €). Pas de spray ni de filtre à remplacer.

## Verdict détaillé

Le Neakasa M1 excelle dans deux domaines : le silence (45 dB, record du comparatif) et l'accueil des grands chats. Si votre Maine Coon de 9 kg refuse d'entrer dans les litières automatiques standard, c'est la solution.

En revanche, l'app est basique — pas de suivi santé ni d'identification multi-chats. Pour les propriétaires qui veulent une expérience connectée poussée, le PETKIT Pura Max 2 ou le Litter-Robot 4 sont plus adaptés.

À 429-475 €, le prix est dans la moyenne du milieu de gamme. La garantie 2 ans pièces et les accessoires inclus (tapis + sacs) rendent le rapport qualité-prix correct, sans être exceptionnel.
