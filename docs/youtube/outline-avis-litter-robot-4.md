### 3. Fonctionnement — Le globe rotatif (2:15 - 3:45)

**Points a couvrir :**
- Le capteur detecte quand le chat sort
- Apres un delai configurable, le globe tourne lentement
- La litiere propre est tamisee, les agglomerats tombent dans le tiroir
- Le globe revient en position initiale — la litiere est nivellee et propre
- Cycle complet en ~2 minutes
- Possible de declencher manuellement depuis le bouton ou l'app
- Fonctionne avec toute litiere agglomerante standard

**Ce qui differencie le globe :**
- Plus efficace que la rotation classique : le tamisage par gravite capture les plus petits agglomerats
- Moins de contact avec les parois = litiere plus propre plus longtemps
- Mecanisme brevete = pas de copie, c'est exclusif Litter-Robot

**Conseil pratique :**
- "Utilisez une litiere agglomerante de bonne qualite. Avec une litiere bas de gamme qui s'effrite, le globe sera moins efficace."
